To help validate the consolidated CYRPRESS-AGENT.md guidance, we plan to create a small ecommerce application.

The application supports the following Application Features:
- A storefront web application that enables a user to:
-- log in
-- browse the store catalog
-- select an catalog item and add it to a shopping cart 
-- check out catalog items
-- add a payment to complete the purchase transaction


- A simple SQLite database that contains:
-- Store catalog with about 20 items preloaded
-- Users tables with 5 users that is able to log into the application

Application Backend:
- A NodeJS-based set of APIs that supports the Application Features described above
- In a folder app/backend
- Runnable via the Cursor Terminal

Application Frontend:
- A REACTJS based Frontend that interacts with the Backend APIs
- In a folder app/frontend
- Runnable via the Cursor Terminal with a browser link
